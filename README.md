# Object-Oriented Programming in JavaScript

This project shows different ways to create and organize objects in JavaScript using OOP concepts.

## What's Inside?

- **classes.js** - Using ES6 classes to create objects (recommended way)
- **factoryFunction.js** - Creating objects with functions that return objects
- **newOperator.js** - Using the `new` operator with constructor functions
- **protoTypes.js** - Understanding prototypes and how to share methods efficiently
- **inheritence.js** - How child classes inherit from parent classes

## Key Concepts

- **Classes** - Templates for creating objects
- **Prototypes** - Share methods without copying them
- **Inheritance** - Child classes can use parent class features
- **Factory Functions** - Alternative way to create objects

All files include examples and explanations to help you learn!
